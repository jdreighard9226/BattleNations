# Battle Nations

A turn-based territory conquest game built in Java, inspired by the board game Risk.

---

## What is it?

Battle Nations is a multiplayer strategy game where players take turns claiming territories,
placing troops, attacking enemies, and fortifying their positions. The first player to meet
the win condition wins the game.

There are two game modes:
- **Total Domination** — control every territory on the map to win
- **Capital Domination** — capture every capital territory to win

---

## How to Run

Make sure you have **Java 17 or later** installed. You can check with:
```
java -version
```

### Mac / Linux

1. Clone the repo
```
git clone https://github.com/your-username/BattleNations.git
cd BattleNations
```
2. Create the output folder
```
mkdir out
```
3. Compile all Java files
```
javac -d out src/**/*.java
```
4. Run the game
```
java -cp out startGUI.StartController
```

## How to Play

**Setup:**
- Pick a map
- Create 2–6 players, each with a unique name and color
- Choose whether territories and troops are assigned randomly or manually
- Select a game mode

**Each turn has 3 phases:**
1. **Reinforce** — place your troops on territories you own
2. **Attack** — attack neighboring enemy territories
3. **Fortify** — move troops between your connected territories

**Combat** is probabilistic — attack and defense strength are calculated using troop counts
and terrain bonuses. The attacker retreats after losing half their starting troops.

## Terrain Types

| Terrain | Attack Bonus | Defense Bonus | Troop Bonus |
|---|---|---|---|
| Plain | 0% | 0% | +0 |
| Desert | +9% | 0% | +0 |
| Mountain | 0% | +10% | +0 |
| City | 0% | 0% | +1 |
| Water | — | — | Not ownable |
| Water Route | — | — | Connects territories |

## Project Structure

```
src/
├── game/          — core game logic (attack, reinforce, fortify services)
├── map/           — territory, region, and world classes
├── player/        — player class
├── terrain/       — terrain types and bonuses
├── gameGUI/       — in-game UI and controller
├── setUpGUI/      — setup phase UI and controllers
├── startGUI/      — main menu, settings, player creation pages
└── soundMaker/    — background music and button sounds
```
